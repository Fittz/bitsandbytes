File description:

Helper functions:

f.m, fInverse: the sigmoid functions used and their inverse
minusPoint5.m, myeigs: helper functions used in network generation

The really important functions:

generateTrainTestData.m: as the name says
generateNet.m: as the name says
learnAndTest.m: the main learning and testing script
continueRun.m: when a network has been trained, use this
    for testing it on novel data
batch.m: a script for an entire experiment, including data
        generation, network generation, learning
